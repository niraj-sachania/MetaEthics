### Design Principles for Audience

The site is designed to:

- **Be informative without being overwhelming** â€“ Content is chunked into digestible sections with clear visual hierarchy

- **Include clear calls to action** â€“ "Report a concern", "Learn more", "Contact help lines" are prominently displayed
- **Maintain a neutral, educational tone** â€“ Accessible for the general public yet credible enough for professionals and educators
